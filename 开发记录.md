**功能规格：**

| 功能 | 说明 |
|------|------|
| **数据加载** | 首次访问 fetch('data/news.json')，完成后缓存到 LocalStorage |
| **缓存策略** | LocalStorage 缓存有效期 1 小时，过期自动清除并重新加载 |
| **冷热加载** | 缓存有效 → 读 LocalStorage（即时显示）；缓存无效/首次 → 发起网络请求 |
| **辅助方法** | getAllCategories() / getAllCompanies() / getAllIssues() / getTotalCount() |

**数据流：**

```
首次打开 → fetch('data/news.json') → 解析 JSON → 渲染页面 → 缓存到 LocalStorage
再次打开 → 读 LocalStorage（缓存有效）→ 跳过网络请求 → 直接渲染
          → 读 LocalStorage（缓存过期）→ 重新请求 news.json → 更新缓存
```
